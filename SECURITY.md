# Security Policy

## Reporting Security Vulnerabilities

If you discover a security vulnerability, please report it by opening an issue in our repository. We will treat any information you provide as confidential and will work to address the vulnerability as quickly as possible.

## Supported Versions Policy

We will provide security updates for the latest two versions of the software. Older versions will not receive security updates once a new version is released. Please refer to the release notes for specific version support details.
